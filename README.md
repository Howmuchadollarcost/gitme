# GitMe

A Github Profiler viewer with charts from ChartJS


Built With:

- Chart.js
- Styled Components
- Github Polyglot
- React Icons
- React spinners

## Getting Started

1. Clone the Repo

    ```
    git clone https://github.com/Howmuchadollarcost/gitme.git
    ```

2. Install dependencies

    ```
    npm install
    ```

3. Start script

    ```
    npm start
    ```
